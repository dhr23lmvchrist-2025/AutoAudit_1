# AutoAudit Performance Visualization System

## Overview
This project provides comprehensive data visualization and machine learning analysis for the AutoAudit system, an AI-powered automated compliance framework for Algorithmic Decision-Making (ADM) systems in Human Resource Management.

## Features
- **Automated Dataset Generation**: Creates realistic synthetic performance data based on research paper metrics
- **Comprehensive Visualizations**: Bar charts, pie charts, complex matrices, line charts, and stacked charts
- **Machine Learning Models**: Predicts performance and improvement based on system characteristics
- **Performance Analysis**: Compares AutoAudit against baseline manual auditing
